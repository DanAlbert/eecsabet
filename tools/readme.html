Administration Tools Help

Navigation to the tools provided with this software may be found at the top of the page. Many of them require you to select a course from the list before they are available. Do so by selecting the link for the appropriate course from the list. Information about each of these tools is available on the page for the tool itself.

Once selected, a course's ABET syllabus information will be displayed at the bottom of the page. To generate a course's ABET syllabus in LaTeX format, click the link above the syllabus preview. To generate ABET syllabi in LaTeX format for all courses offered this term, use the 'Dump LaTeX Syllabi for All Current Courses' tool.
