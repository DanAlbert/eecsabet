Administration Tools Help

Navigation to the tools provided with this software may be found at the top of the page. Many of them require you to select a course from the list before they are available. Do so by selecting the link for the appropriate course from the list. Information about each of these tools is available on the page for the tool itself.

Once selected, a course's ABET syllabus information will be displayed at the bottom of the page. To generate a course's ABET syllabus in LaTeX format, click the link above the syllabus preview. To generate ABET syllabi in LaTeX format for all courses offered this term, use the 'Dump LaTeX Syllabi for All Current Courses' tool.

Checklist for Creating a Course

There are a few tasks you must complete before an instructor will be able to effectively use this software. The action that will bring them into the process is the creation of their course instances.

A note on course instances

A course instance is an offering of a course for a given term and instructor. If more than one instructor is teaching a course in a given term, you will need to create multiple course instances. However, if one instructor is theaching more than one section of a course in a given term, you only need to create one course instance.

Before creating a course instance

You should not create a course instance until all of the required course information has been provided. CLOs, prerequisites, course content, learning resources, and the terms that a course is offered make up the required course information. Below is the recommended order in which you provide course information. Once you begin with Create a Course, you will be guided through the process by each page.

  1. Create a Course
  2. Create and Modify CLOs
  3. Add Course Content
  4. Add Learning Resources
  5. Select Terms this Course is Offered
  6. Select Course Prerequisites

A note on prerequisites

A course needs to exist in the database before it can be added as a prerequisite. For this reason, it is best to work your way up from the lower level courses. If you need to add a course that will not be used beyond its participation as a prerequisite (ex. MTH 256), you should create the prerequisite course and skip the rest of its configuration.

Creating a course instance

Once you have created the course and filled out all of the required information, ensure that you have added the appropriate instructors and then create the course instances. Once you create the course instance, the instructor will be mailed a link to the assessment page for their course.
