Term State Tools Help

Nagging Instructors

Any instructors that have yet to fill out their ABET forms will be listed. To email reminders to the listed instructors, click 'Nag'.

Changing Course States

If you wish to ready all courses in the current term for finalization, click 'Ready Courses for Finalization'. Note that you do not need to do this, as it will be done automatically around week eight of every term.
