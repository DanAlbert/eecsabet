Prerequisite Tools Help

Adding New Prerequisite

Select a prerequisite from the list. If the course is a corequisite, check the 'Is Corequisit' box. Any prerequisites that have already been added to the course will be listed under Alternatives. If any of these courses are alternatives to the course you are current adding, check the box next to the course's title. When finished, click 'Create New Prerequisite'. If you need to edit information about a prerequisite, remove it and recreate it.

Removing Prerequisites

Check all of the prerequisites you would like to remove and click 'Submit'.
