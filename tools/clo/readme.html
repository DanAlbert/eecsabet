CLO Tools Help

Adding a New CLO to a Course

Fill out the Description and ABET Outcomes fields and click 'Create New CLO'. If you need to edit a CLO, delete it and recrate it.

Removing a CLO From a Course

Check all of the CLOs you would like to remove and click 'Submit'.

Reordering CLOs

Use the + and - buttons to move a CLO up or down, then click 'Submit'.
